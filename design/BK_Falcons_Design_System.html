Design System & UI Kit
v1.0 · Feb 2026 · Mobile-First
Overview
This design system defines the visual language for the Brooklyn Falcons website redesign. All tokens, components, and patterns are derived from two sources: the existing brand identity on brooklynfalcons.com and the UX research findings from 22 respondents. The system is mobile-first (420px base) and designed for a Wix-to-modern-platform migration.
Design Principles

1. Two Journeys, One Brand — Community programs and travel ball share a visual system but have distinct entry points and tones.
2. Mobile-First, Always — 100% of parents/players use smartphones. Every decision starts at 420px.
3. Low-Maintenance Freshness — Components auto-populate from GameChanger, newsletters, and calendars to prevent content staleness.
4. Trust Through Clarity — Clear navigation, visible costs, and prominent CTAs build confidence for first-time visitors.

Brand Voice

Tagline: "Nothing is Given. Compete and Earn."
Tone: Confident but welcoming. The site speaks to parents first (practical, trustworthy) and players second (aspirational, energetic).
Key words: Family, development, exposure, compete, opportunity, commitment.

5. Layout Patterns
Repeatable page structures that create consistent rhythm across the site. These patterns emerged from wireframing 7 pages and map directly to how users scan content on mobile.
PatternStructureUsed OnPurpose
section padding: 16px, white bg, label + title + content All pages Standard content block with side padding. Creates breathing room.
section-flush Full-width, no horizontal padding, content inset individually All pages Card grids, info grids, photo galleries. Edge-to-edge visual weight.
Dark hero Full-width dark bg, large title, subtitle, optional photo overlay Homepage, Travel, Alumni, Newsletter Page-level identity block. Sets tone immediately on landing.
2×2 info grid grid-template-columns: 1fr 1fr, info cards 10U Detail, Travel Detail At-a-glance program metadata (season, cost, location, schedule).
Horizontal scroll overflow-x: auto, flex row, snap points Homepage (ticker, age groups, testimonials) Mobile-native browsing for sequential content. Avoids long vertical lists.
Card grid 2-col or 3-col responsive grid of uniform cards Newsletter archive, Alumni, Community programs Browsable collections (articles, stories, programs).
Stat row 3-across centered numbers with labels below Alumni, Travel Ball Impact metrics (college commitments, years, players developed).
Pipeline nudge Left-bordered callout card, light bg, link arrow Community → Travel, Travel → Alumni Cross-journey discovery. Research found parents don't know pipeline exists.
Breadcrumb bar Light gray bg strip, Home > Section > Page 10U Detail, individual articles Wayfinding for deep pages. Supports back-navigation on mobile.
Newsletter promo strip Full-width dark bg, title + description + email input Homepage, Newsletter, Community, Travel Appears before footer on content pages. Drives email list growth.

7. Accessibility & Contrast
All color combinations meet WCAG 2.1 AA standards. Key contrast ratios validated below.
CombinationRatioPasses
neutral-900 on neutral-0 (body text)~16:1✅ AAA
neutral-0 on neutral-900 (dark hero text)~16:1✅ AAA
falcon-600 on neutral-0 (section labels)~5.2:1✅ AA
falcon-900 on falcon-200 (primary button)~9.6:1✅ AAA
neutral-600 on neutral-0 (secondary text)~5.7:1✅ AA
gold-400 on neutral-900 (ticker scores)~9.8:1✅ AAA
neutral-0 on falcon-600 (dark button)~5.9:1✅ AA
Note: falcon-200 (#A3D3ED) is the primary UI accent. It uses dark text (falcon-900) for buttons, achieving 9.6:1 AAA contrast. falcon-200 on white is only 1.6:1, so it works for decorative borders and backgrounds only — never for text. For text on light backgrounds, use falcon-600 (#2E6F97) minimum at 5.5:1 AA.
